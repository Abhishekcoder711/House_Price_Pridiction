# 🏠 House Price Prediction using Multiple Linear Regression

## 📌 Overview

This project focuses on predicting house prices using a machine learning approach. A **Multiple Linear Regression model** is implemented to estimate property prices based on key features such as carpet area and number of bathrooms.

The project demonstrates the complete machine learning workflow including data preprocessing, model training, prediction, and evaluation.

---

## 🚀 Features

* Predicts house prices based on multiple input features
* Uses **Multiple Linear Regression** for better accuracy
* Compares **actual vs predicted values**
* Evaluates model performance using **R² score**
* Simple and easy-to-understand implementation

---

## 🧠 Machine Learning Approach

* Supervised Learning
* Multiple Linear Regression

---

## 📊 Dataset

The dataset contains housing-related information such as:

* Carpet Area
* Number of Bathrooms
* Price (Target Variable)

---

## ⚙️ Workflow

1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split (70:30)
5. Model Training
6. Prediction
7. Model Evaluation

---

## 📈 Model Evaluation

The model performance is evaluated using:

* **R² Score** (to measure accuracy of predictions)
* Comparison between **Actual vs Predicted values**

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

---

## 📌 Results

The model successfully predicts house prices based on multiple features and shows a good fit between actual and predicted values.

---

## 📷 Output Sample

Displays predicted values alongside actual values for better understanding of model performance.

## 📬 Conclusion

This project demonstrates how regression techniques can be applied to real-world problems like housing price estimation. It also highlights the importance of feature selection and model evaluation in machine learning.
